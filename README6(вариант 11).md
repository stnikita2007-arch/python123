Имитация отжига (Simulated Annealing)

Принцип работы алгоритма

Имитация отжига — это метаэвристический алгоритм оптимизации, вдохновленный процессом отжига в металлургии, где металл нагревается и медленно охлаждается для уменьшения дефектов кристаллической решетки.

Основные этапы алгоритма:

Инициализация:

Начальное решение (состояние)
Начальная температура T
Параметры охлаждения
Итерационный процесс:

text
Пока T > T_min:
    Для k итераций при текущей температуре:
        - Генерируем соседнее решение
        - Вычисляем разницу энергии ΔE = E(новое) - E(текущее)
        - Если ΔE > 0 (улучшение): принимаем новое решение
        - Иначе: принимаем с вероятностью exp(ΔE / T)
    
    - Уменьшаем температуру по схеме охлаждения
Критерий остановки:

Достижение минимальной температуры
Превышение максимального числа итераций
Отсутствие улучшений
Роль параметра температуры

Контроль вероятности принятия худших решений

Формула вероятности принятия:

text
P(принятия) = exp(ΔE / T), где ΔE < 0
Влияние температуры:

Высокая температура (начальная фаза):

Большая вероятность принятия худших решений
Широкий поиск в пространстве решений
Исследование различных областей
Низкая температура (конечная фаза):

Малая вероятность принятия худших решений
Локальный поиск вблизи текущего решения
Эксплуатация найденных хороших решений
Схемы охлаждения

Геометрическое охлаждение:

text
T_{k+1} = α × T_k, где 0 < α < 1
Линейное охлаждение:

text
T_{k+1} = T_k - β
Логарифмическое охлаждение:

text
T_k = C / log(1 + k)
Преимущества имитации отжига

Способность избегать локальных оптимумов благодаря принятию худших решений
Простота реализации
Универсальность - применим к различным задачам оптимизации
Гарантия сходимости к глобальному оптимуму при правильных параметрах
